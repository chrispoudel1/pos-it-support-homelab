# POS IT Support Homelab

## Overview
This project simulates a real-world retail and payments IT support environment, designed to mirror Level 1 / Junior IT Support responsibilities.

The lab focuses on:
- POS system support
- Server connectivity
- Network troubleshooting
- Firewall configuration
- Incident documentation and escalation

## Lab Architecture
- POS-Terminal-01 (Windows 11)
- Server-01 (Application / File Server)
- pfSense Firewall
- Simulated retail network (LAN/WAN separation)

## Technologies Used
- Oracle VirtualBox
- Windows 11
- pfSense
- Square POS (demo environment)
- CSV transaction reporting
- Basic networking (IP, DNS, NAT)

## Key Scenarios Simulated
- POS unable to connect to backend server
- Network connectivity issues
- Application-level troubleshooting
- Transaction verification and reporting
- Incident documentation and follow-up
